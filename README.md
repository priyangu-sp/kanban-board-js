# Kanban Board (VanillaJS)

A dynamic Kanban board built using HTML, CSS, and JavaScript. Users can add, delete, and drag tasks between columns. All tasks are saved and persist even after reloading the page.

## Features

* Add tasks to the To-Do column
* Drag-and-drop tasks between To-Do, In Progress, and Completed columns
* Delete tasks with a button
* Tasks are saved and persist across page reloads

## Tech Stack

* HTML
* CSS
* Vanilla JS
